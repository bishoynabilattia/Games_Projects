# WordQuest AI 🎮🤖

WordQuest AI is an interactive Python application built with Tkinter that solves word search puzzles using AI search algorithms. The project visualizes how different search strategies explore a grid of letters to find a specific target word.

---

## ✨ Features

* **Visual Maze/Grid:** A $10 \times 10$ graphical grid generated dynamically with Tkinter.
* **Algorithm Visualization:** Watch how different search algorithms explore the paths in real-time.
* **Path Highlighting:** Animates the exploration process and highlights the final correct path in **Gold** 🏆 upon success.
* **Interactive Controls:** Mouse bindings to manually trigger, click, or reset cell colors.

---

## 🚀 Supported Algorithms

The project demonstrates and compares three core search algorithms:
1. **Depth-First Search (DFS):** Explores as deep as possible along each branch before backtracking (Visualized in Orange/Green).
2. **Breadth-First Search (BFS):** Explores all neighbor nodes at the present depth before moving to the next level (Visualized in White/Red).
3. **A* Search (Informed):** Uses a Manhattan distance heuristic to find the most optimal path efficiently (Visualized in Green/Cyan).
